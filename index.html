Projects

Life On Mars

As part of a team of five data engineers we created an ETL pipeline to look at exoplanets and attempt to estimate the probability that life could exist on some of these planets. The team found a suitable data source at the NASA Exoplanet Archive and setup a TAP request to extract relevant features for our study. The main steps in this project were:

  • EDA of the dataset
  • Creation of an ETL pipeline using Microsoft Azure cloud services
  • Creation of an interactive Power BI dashboard
  • Creation of a Life On Mars website
  • Creation of a Life On Mars promotional video

Credit Rating Calculator

For module seven of my Masters, I was examining an imbalanced classification dataset set on credit issued to customers of a bank. Depending on the credit rating of the customer the loaned sum was either returned to the bank with interest or the customer defaulted on the loan. The goal of the project was to use machine learning to provide a recommendation as to whether the bank should provide a loan to a particular customer or not. The main steps in this project were:

  • EDA of the dataset
  • Preparation of the imbalanced data with SMOTE and undersampling
  • Comparison of several machine learning models scored on accuracy and speed
  • Understanding of the most important features with SHAP and LIME
  • For the highest scoring models hyperparameter optimisation with a custom scoring function
  • Creation of a Credit Rating Calculator website

Big Data with Spark

For module eight of my Masters, I was preprocessing image files with ResNet50 in Spark on an AWS EMR instance. The main steps in this project were:

  • Transfer of the image dataset to an S3 bucket
  • Creation of an array of ResNes50 image vectors
  • Performing a PCA on the ResNes50 image vectors
  • Saving the PCA features to the S3 bucket
